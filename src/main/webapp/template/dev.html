Share our food

Post GET: /api/foodreview/post?(text=&page=) 
            Get post(s) by text and page
            QueryParam: -text optional default null
                        -page optional default 1
            Example: /api/foodreview/post?(text=banana&page=2)
GET: /api/foodreview/post/{id} 
            Get post by id
            QueryParam: -id required
            Example: /api/foodreview/post/157
Comment GET: /api/foodreview/comment/{post_id} 
            Get all comments by post id
            QueryParam: -post_id required
            Example: /api/foodreview/comment/157
POST: /api/foodreview/comment/{post_id} 
            Insert one comment to post has specified id
            QueryParam: -post_id required
            RequestBody:
                {
                    "content": "Very delicious!",
                    "name": "Leo",
                    "contact": "leo@abc.com"
                }
            Example: /api/foodreview/comment/157
